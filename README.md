# TelecomX LATAM — Predicción de Churn

**Autor:** Maycol Ninaja

## Descripción
Proyecto compacto para predecir abandono de clientes (churn) mediante análisis de datos y modelos de machine learning.

## Objetivos
- Identificar factores asociados al churn.  
- Construir y comparar modelos predictivos (Random Forest, KNN).  
- Detectar clientes de alto riesgo para acciones de retención.

## Archivos principales
- `ChallengeTelecomXParteII.ipynb`  
- `TelecomX_Datos_limpio.csv`

## Tecnologías
Python, pandas, NumPy, scikit-learn, imbalanced-learn (SMOTE), matplotlib, seaborn, Jupyter
